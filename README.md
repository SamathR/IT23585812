#Assignment 1 - Singlish to Sinhala Translator Testing

This repository contains automated Playwright test cases for testing the Singlish to Sinhala translation system.

# Tools Used
- Playwright
- JavaScript
- Node.js
- https://www.swifttranslator.com/

# Project Overview
The object of assignment is to :
      Desing positive and Negative cases
      Automate test execution using "Playwright"
      Evaluate the accuracy and robustness of the singlish to sinhala translator
      Document test result using an Excel test case sheet

## Test Coverage
- Positive Functional Test Cases
          Greeting and day-to-day expressions
          Present, past, and future tense sentences
          Polite requests and imperative commands
          Negative form sentences
          Compound and long sentences
          Mixed Singlish and English words
          Brand names, technical terms, numbers, and units
- Negative Functional Test Cases
          Random symbols input
          Emoji mixed input
          Numeric-only input
          Meaningless text strings
          Excessive character repetition
          Very long input strings
          Broken sentence structures
  
- UI Test Cases
          Real-time output updates while typing
          Input field behavior
